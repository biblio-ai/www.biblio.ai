Features

Image Metadata Enrichment

Using state-of-the-art computer vision technologies from Amazon AWS and Microsoft Azure to enrich and bring new depth and possibilities to the Library's digital image collection.

Automatic cataloguing

Automatic text recognition and transciption of historical handwritten documents (maps, jounrals, notes..), Object detection to get the identify and tag thousands of objects within an image. Identify image types and colour schemes in pictures. Recognize brands, celebrities and landmarks. Automatically catalogue the gender and age of people within image.

Accessibility

Enable vision impaired users greater accessibility to your image collection. Existing image repository records may have limited descriptive information which disadvantages vision impaired users when trying to understand, search and discovery visual content in your collection

Distribution

Biblio AI works with your existing Library Discovery Layer, library content aggregators (Primo Central Index ..), and APIs ready for search engine distribution

Discovery and Re-use

Not limited to classic library search platforms, Biblio AI can distribute your content directly to creator centric services to aid content re-use by non-traditional library users

100% Open Source

Totally open-source witn no vendor lock-in and not tied to any specific services or providers. Any image repository provider, any library discover layer and whichever cloud/hybrid/on-prem architecture that suit you best

No change required

Biblio AI is a drop in solution where no change is required to you existing image repository.

Platform netural

Biblio AI works with any image repository or library discovery layer that implements the repository interoperability standard OAI-PMH

AI enable legacy systems

Got a legacy image repository with no upgrade path? Biblio AI is a drop-in complimentary service that auto AI enables your OAI-PHM service
